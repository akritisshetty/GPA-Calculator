# GPA-Calculator
A simple web-based GPA Calculator to help students calculate their SGPA and CGPA based on grades and credits. It allows users to input subjects, grades, and credits, calculate GPA, and generate printable reports.
## Features:
1. SGPA Calculator: Add subjects, assign grades (O, A, B, C, D, F), input credits, and calculate SGPA.
2. CGPA Calculator: Enter SGPA for each semester and calculate CGPA.
3. Edit & Delete Options: Modify or remove subjects and semesters.
4. Printable Report: Generate a clean report for SGPA and CGPA.
5. Responsive Design: Works seamlessly on desktop and mobile devices.

## Technologies Used:
- HTML5, CSS3, JavaScript

## How to Use:
1. Clone the repository.
2. Open calc.html in your browser.
3. Enter name, select SGPA or CGPA calculator, input details, and calculate.
4. Print the results with the Print Report button.
